Javascript & jquery

This flash cards contains meaning of various programming language.

  • JavaScript is a high-level programming language primarily known for its use in web development. It is often described as the language of the web because it allows developers to create interactive and dynamic content on websites.
  • jQuery, on the other hand, is a fast, small, and feature-rich JavaScript library. It simplifies many tasks in JavaScript such as HTML document traversal and manipulation, event handling, and animation. jQuery aims to make it easier to navigate a document, select DOM elements, create animations, handle events, and develop Ajax applications.

  • In JavaScript, functions are blocks of reusable code that perform a specific task or calculation. They encapsulate a set of statements that can be invoked multiple times with different inputs, producing a desired output. Functions are defined using the function keyword followed by a name (optional) and a set of parameters enclosed in parentheses. They can optionally return a value using the return statement.
